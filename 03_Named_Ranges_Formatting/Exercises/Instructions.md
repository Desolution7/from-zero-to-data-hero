# Milestone 3 – Exercises 01 & 02

This assignment covers both Named Ranges and Worksheet Formatting concepts.

---

## Exercise 01 – Named Ranges

### Part A: Naming Ranges

Name the following data ranges:

- A2:A52 → Customer_Name
- B2:B52 → Quantity
- C2:C52 → Price_Per_Item
- D2:D52 → Total
- E2:E52 → Sales_Tax
- F2:F52 → Total_Inc_Tax
- H2:I2 → Tax_Rate

> Different methods were used (Name Box, Define Name, Create from Selection).

### Part B: Using Named Ranges in Formulas

1. Replace the reference to cell I2 in the Sales Tax formula (Column E) with the named range `Tax_Rate`.
2. Recalculate formulas in cells I4:I8 using named ranges instead of direct cell references.

---

## Exercise 02 – Number Formatting and Worksheet Formatting

### Applying Number Formats

1. Apply Short Date format to column A.
2. Apply Accounting format to column C.
3. Apply Percentage format to the value in cell 14.
4. Use a formula to complete the Sales Tax in column D and apply Accounting format.

### Applying Worksheet Formatting

1. Format the worksheet title:
   - Bold
   - Light Grey background
   - Font size 14pt

2. Format column headings:
   - Bold
   - Light Green background

3. Apply borders to the dataset:
   - All Borders
   - Dark Green color
