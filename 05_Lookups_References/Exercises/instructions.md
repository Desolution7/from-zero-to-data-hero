# Milestone 5 – Lookup Functions (Exercise)

## Objective
To apply lookup functions in Excel to dynamically retrieve related data based on user selection.

---

## Dataset Overview
A dataset was provided containing the following columns:

- Athlete Name
- Bib Number
- Route
- Position

A separate lookup section was created to display selected athlete details dynamically.

---

## Tasks

### 1. Create Dropdown Selection
- Create a data validation dropdown list in **cell H3**
- The dropdown should contain all athlete names from the dataset

---

### 2. Implement Lookup Function
Using a lookup function of your choice (e.g., VLOOKUP, INDEX & MATCH, or XLOOKUP):

- Retrieve the following details based on the selected athlete:
  - Bib Number
  - Route
  - Position

- Display results dynamically in the lookup section (column H)

---

### 3. Add Error Handling
- Implement error handling (e.g., `IFERROR`)
- Ensure the formula handles cases where no selection is made or data is missing
- Prevent display of `#N/A` errors

---

## Outcome

- Built a dynamic lookup system using Excel functions  
- Applied data validation for controlled user input  
- Implemented error handling for improved user experience  
- Demonstrated practical use of lookup functions for real-world data retrieval
